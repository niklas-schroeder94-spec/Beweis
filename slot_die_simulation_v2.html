This tool is an engineering screening model. All scores are heuristic plausibility estimates, not physical certainty. They are based on dimensionless numbers and film geometry thresholds established from coating engineering literature and practical experience.
Physics Equations
Mass and Film Balance: m_wet = m_dry / x_s [g/m²] h_wet = (m_wet / 1000) / rho_wet [m] h_dry = (m_dry / 1000) / rho_dry [m] Volumetric Flow Rate: Q = U · b · h_wet [m³/s] Q_L_min = Q · 60000 [L/min] Dimensionless Numbers: Ca = mu · U / sigma Capillary number Re = rho_wet · U · H / mu Reynolds number Lambda = H / h_wet Gap ratio gamma_dot = U / H Shear rate [1/s] Pressure Scale: delta_p = 12 · mu · U · L / H² [Pa] Penetration / Effective Thickness: h_pen(t) = k_pen · sqrt(t) [m] (optional) h_eff_wet = h_wet · (1 - pen_frac) - h_pen(t) h_eff_dry = h_dry · (h_eff_wet / h_wet) Continuity Reserve: reserve = h_eff_dry / min_continuous_dry_threshold
Viscosity Models
Mode A — Direct Input: mu = mu_direct [Pa·s] Mode B — Exponential Thickener Model: expo = k_thick · c_thick + k_solids · (x_s - x_s_ref) mu = mu_base · exp(expo) [Pa·s] Mode C — Power-Law: K = mu_base · exp(expo) mu(gamma_dot) = K · gamma_dot^(n-1) [Pa·s] Parameter Correlations (Monte Carlo): Higher solids → mu increases (positive correlation) Higher mu → penetration decreases slightly Lower sigma → improved wetting (Ca risk reduced)

Model Assumptions & Limitations
  • No CFD simulation — flow inside die slot is not resolved
  • No explicit air entrainment velocity threshold calculation
  • No microdefect / pinhol / morphology prediction
  • Barrier resistance / diffusion not simulated
  • Penetration model is static fraction (or simple sqrt(t)); no capillary transport model
  • Scoring bands are engineering heuristics, not physical limits
  • Correlated MC uses simplified linear correlation approximation
  • Density is assumed constant (no composition-viscosity iteration)
  • Output is plausibility screening for process window assessment only
