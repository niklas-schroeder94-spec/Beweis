Physikalische Gleichungen
Massen- und Filmbilanz: m_wet = m_dry / x_s [g/m²] h_wet = (m_wet / 1000) / rho_wet [m] h_dry = (m_dry / 1000) / rho_dry [m] Volumenstrom: Q = U · b · h_wet [m³/s] Q_L_min = Q · 60000 [L/min] Kennzahlen: Ca = mu · U / sigma Re = rho_wet · U · H / mu Lambda = H / h_wet gamma_dot = U / H Druckmaß: delta_p = 12 · mu · U · L / H² [Pa] Effektive Schicht: h_pen(t) = k_pen · sqrt(t) h_eff_wet = h_wet · (1 - pen_frac) - h_pen(t) h_eff_dry = h_dry · (h_eff_wet / h_wet) Reserve: reserve = h_eff_dry / min_continuous_dry_threshold
Viskositätsmodelle
Modus A — Direkte Eingabe: mu = mu_direct Modus B — Verdicker-Modell: expo = k_thick · c_thick + k_solids · (x_s - x_s_ref) mu = mu_base · exp(expo) Modus C — Potenzgesetz: K = mu_base · exp(expo) mu(gamma_dot) = K · gamma_dot^(n-1) Korrelationen in Monte Carlo: Höherer Feststoff -> höhere Viskosität Höhere Viskosität -> etwas geringere Penetration
